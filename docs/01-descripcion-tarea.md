# 1. Descripción de la tarea

Este proyecto, desarrollado en el marco del Fondo Social Europeo, consiste en la creación y prueba de una aplicación web para la gestión de subastas. El caso práctico se centra en la colaboración de Ada, María y Juan, donde tras la fase de diseño, se aborda el desarrollo y la fase de pruebas.

El objetivo es construir una solución robusta, intuitiva y funcional, siguiendo buenas prácticas de desarrollo y asegurando la calidad mediante pruebas unitarias e integración. Se busca que la aplicación sea fácilmente instalable tanto en local como en hosting online, cumpliendo con los requisitos de la tarea y permitiendo la gestión segura de usuarios y subastas.

# 2. Información de interés

## Tecnologías utilizadas
- **Frontend:** Angular 18, HTML5, CSS3
- **Backend:** .NET 8 WebAPI
- **Base de datos:** SQL Server (compatible con MySQL)
- **Otros:** Entity Framework Core, JWT, Azure Static Web Apps

## Justificación de elección
- Angular y .NET ofrecen documentación extensa y comunidad activa.
- Permiten reutilizar código y componentes previos.
- Facilitan la integración y despliegue en Azure y otros hostings.

## Reutilización y documentación
- Se han aprovechado librerías y patrones de proyectos anteriores.
- La documentación generada cubre instalación, uso y pruebas.

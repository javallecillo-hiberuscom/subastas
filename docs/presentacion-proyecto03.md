## 5. Tecnologías empleadas
- Backend: .NET 8 WebAPI
- Frontend: Angular 18, HTML5, CSS3
- Base de datos: SQL Server (compatible MySQL)
- Pruebas: xUnit (.NET), equivalente a JUnit
- Control de versiones: Git
- Hosting: Azure Static Web Apps/App Service
- [Logos de tecnologías]

## 6. Justificación tecnológica
- .NET y Angular: robustez, comunidad, documentación
- SQL Server: integración nativa, escalabilidad
- xUnit: estándar en .NET, pruebas automáticas
- Compatibilidad con MySQL para flexibilidad
